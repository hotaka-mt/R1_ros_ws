# ロボコン2024Bチーム ロボット1「鶴明」ReadMe
## ロボット構成
### アクチュエータ
* メカナム4輪
* サーボ×3∼4(ラッチ開閉用)
* MD×1(発射機構再装填用)
### センサ
* RealeSense(自己位置)
* OTOS(自己位置)
* ロリコン×2（自己位置）
* リミット×4(自己位置)
* リミット×2(発射機構再装填用)
